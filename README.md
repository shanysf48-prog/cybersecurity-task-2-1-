# Phishing Email Analysis - Task 2

## 📚 Internship Task
Cyber Security Internship – Task 2

## 🎯 Objective
Analyze a suspicious email sample and identify phishing characteristics.

---

## 🔍 Phishing Indicators Found

1. Suspicious sender email address (spoofed domain)
2. Urgent/threatening language used
3. Suspicious link (mismatched URL)
4. Spelling and grammar errors
5. Request for sensitive information
6. Header discrepancies found using online analyzer

---

## 🛠 Tools Used
- Email client
- Free online email header analyzer

---

## 📖 What is Phishing?
Phishing is a cyber attack where attackers impersonate trusted entities to steal sensitive information like passwords, credit card details, etc.

---

## 🚨 Why Phishing is Dangerous?
- Data theft
- Financial loss
- Identity theft
- Malware infection
